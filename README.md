# Dotfiles

This repository contains my personal dotfiles and configuration files for a Hyprland-based Linux desktop environment. It includes configurations for window manager, terminal, editor, and various utilities.

## Features

- **Hyprland**: Tiling window manager configuration
- **Waybar**: Highly customizable status bar with multiple themes
- **Kitty**: Terminal emulator with custom theme
- **Neovim**: Text editor with LazyVim-based configuration
- **Rofi**: Application launcher with various themes
- **Swaylock/Swaync**: Screen locker and notification center
- **Wofi**: Alternative launcher
- **Cava**: Audio visualizer
- **Lazygit**: Git TUI

## Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/Racketlon/dotfiles ~/.config
   ```

2. Install required dependencies:
   - Hyprland
   - Waybar
   - Kitty
   - Neovim
   - Rofi
   - Swaylock/Swaync
   - Wofi
   - Cava
   - Lazygit

3. Install fonts:
   - JetBrains Mono Nerd Font
   - Other fonts as needed

4. Restart your session or reload configurations.

## Usage

- The main configuration files are in their respective directories.
- Waybar has multiple themes - switch using the provided scripts.
- Rofi has various themes and configurations for different use cases.
- Neovim is configured for both terminal and VSCode usage.

## Scripts

- `hypr/scripts/`: Various utility scripts for Hyprland
- `waybar/scripts/`: Waybar-related scripts

## Customization

Feel free to modify the configurations to suit your needs. The repository is organized by application for easy navigation.
